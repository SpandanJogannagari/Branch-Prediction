
Step1: Keep sim.py, smith.py ,bimodal.py, gshare.py, hybrid.py including with the trace files in a single directory and change your present working directory to directory where we saved the above files.

Step 2: Now use the below command to run the program in the command prompt .

Command: Python3 sim.py <arguements>
Example: 
python3 sim.py smith 3 gcc_trace.txt 
python3 sim.py bimodal 6 gcc_trace.txt 
python3 sim.py gshare 9 3 gcc_trace.txt 
python3 sim.py hybrid 8 14 10 5 gcc_trace.txt
